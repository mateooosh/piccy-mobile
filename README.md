# Piccy - mobile
Photo sharing social networking service - mobile version

## Technologies
 Project is created with:
* React 16.13.1
* React Redux 7.2.4
* React Native 0.63.4
* React Navigation 5
* Expo 41.0.1
* Native Base 3.0.6
* Socket.io 4.1.2
